VScode安装教程

工具 前端编辑器 2020/01/06

软件下载安装

  1）Ctrl + 点击进入 VSCode官网

  2）点击Download for Windows

配置中文插件

  1） Vscode是一款开源的跨平台编辑器。默认情况下，vscode使用的语言为英文(en)；

  2）VScode使用快捷键组合【Ctrl+Shift+p】

  3）输入Configure Display Language,选择en,点击后会提示重启:选择Restart重启；

  4）重启后如下:点击左边小方框 后输入chinese,点击第一个install,之后提示yes就好啦!

  5）有不一样的情况请自行参考百度解决。

VScode快捷键

编号 快捷键 定义
1 F2 重命名
2 Ctrl + B 侧边栏打开关闭
3 Alt + Shift 块选择
4 Alt + Shift + F 格式化代码
5 Ctrl + Shift + 【 或者 Ctrl + Shift + 】 展开折叠代码折叠
6 Ctrl + K 再按 Ctrl + 0 折叠所有代码
7 Ctrl + K 再按 Ctrl + J 展开所有代码
8 Ctrl + D 选中相同单词
9 Alt + Shift + ⬆(⬇) 向上或向下复制一行代码
10 Alt + Ctrl + ⬆(⬇) 向上或向下添加光标
11 Ctrl + Shift + ⬆(⬇) 向上或向下选中代码
12 Ctrl + Shift + ←(→) 向左或向右选中代码
13 Ctrl + G （输入数值）或者 Ctrl + P （输入:数值） 跳转到指定行号
12 Shift + Delete 向下删除行
13 Ctrl + Delete 向下删除
14 Ctrl + Backspace 向上删除
